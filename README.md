# Ground-O-Mall
collection of factorio blueprint strings in reference to the 'Ground-O-Mall' project



# How to use
select a blueprint version of your choice, the one ending with "-vanilla" doesn't require any mods, the one ending with "-textplates" only works when Textplates are installed (get here: https://mods.factorio.com/mod/textplates)

after you made your decision, copy the entire text inside the file, tab into factorio and click 'import string' on the hotbar.
paste the content into the text box and BOOM , Blueprint.
